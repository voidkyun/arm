This is the repository for ARM - Algebra Relational Mapping, an open-source framework for Web APIs.

Initial idea: `arm.md`

To be a globally recognized open-source project, the natural language within the repository will be consistently English.

However, responses to users should always be in Japanese.

## Dev Rule

When you implement a change, commit it to the appropriate branch unless otherwise instructed, and if it's an implementation for an issue, create a pull request as well.
